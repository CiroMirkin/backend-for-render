# Notes App - Backend Practice Project


## Dev

> Create the `.env` file and fill in the data according to the `example.env` file.

Run at the same time:

In *frontend* folder:

```bash
npm run dev
```

In *backend* folder:

```bash
npm run start
```

For run backend test:
```bash
cd backend
npm run test
```
